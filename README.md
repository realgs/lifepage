## Installation

Clone the repo and install dependencies.

```bash
$ git clone https://github.com/appalaszynski/lifepage.git
$ cd lifepage
$ npm install
```

---

## Usage

### Running Development Server

```bash
$ npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) to see your app.

### Deploying Locally

```bash
$ npm run build
```
